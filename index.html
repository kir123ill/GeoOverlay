1. Снимок

Свойства снимка
Папку с фото достаточно выбрать один раз в текущей сессии. Сам файл изображения используется только для привязки и предпросмотра, но не записывается в JSON. В JSON сохраняются точки, привязка, области, комментарии, метаданные и название файла снимка для поиска в data/tiles. Для готовой базы JSON лежат в папке data/regions, а фото — в папке data/tiles.

Снимок для привязки

Открыт сохранённый файл АФС в режиме просмотра. Контрольные и проверочные области скрыты.
Снимок и точки этого режима хранятся отдельно от «Моего проекта» и никогда не включаются в основной JSON проекта.

Поворот меняет только рабочее отображение снимка перед расстановкой точек. Координаты точек сохраняются относительно исходного файла, поэтому привязка и внешний файл изображения остаются совместимыми.
Распределяйте опорные точки по всей площади снимка, особенно по углам и краям. Точки, расположенные только в центре или на одной линии, дают нестабильную привязку.
